A project dedicated to exploring association rule mining using the Apriori algorithm.

Learn Apriori

The Apriori algorithm is used for mining frequent itemsets and relevant association rules. It operates on databases containing transactions.

How to use this tool

  1. Upload Dataset: Drag and drop your CSV dataset containing transactions. Ensure the first row has column headers.
  2. Select Fields: Choose the specific fields/columns from the dataset that you want to analyze.
  3. Adjust Parameters: Set the minimum support (how frequently an itemset must appear) and confidence (how reliable the rule must be).
  4. Discover: Click the button to run the algorithm and view your frequent itemsets and association rules.
